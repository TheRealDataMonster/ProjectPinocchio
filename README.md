# ProjectPinocchio
Making a real boy... Or a Pet... Or anything that has a soul

## Goal
The goal of this project is to ideate and experiment on how different types of context and sophisticated logic around AI and its physical form (a robot), may be able to emulate an unique life-like experience. Applications of interest are artistic as well as entertainment focused. Some utility driven experiments may be done to determine what types of information need to be incorporated to the "soul" to create a life-like experiences that have real-world impact beyond "putting on a show". It's really about figureing out a way to bring faimiliarity to technology.

## Based On 
This project has its start forking 2 existing repos.
1) https://github.com/pollen-robotics/reachy_mini_conversation_app - by pollen-robotics team. This repo is meant to be forked then some significant changes are meant to be made starting with adding options for variety of models then how it handles the "personality" of the robot.

2) https://github.com/arc-megicxyz-bot/arc-soul - as an inspiration for how to structure variety of contextual information and additional context that it must create based on the information that is recorded - idenity of the "soul", its origins, how and what it remembers, how self-aware it is, etc... This documentation and the way that the AI application handles the series of docs will continue to be more and more important as it improves upon porevious versions.
